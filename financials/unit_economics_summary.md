# Unit Economics Summary

## Product

Gluten-Free Multi-Grain Flour by Aljannat Foods  
Pack size: 1 kg resealable pouch  
Selling price: Rs. 999 per kg  

## Per-Pouch Economics

| Metric | Value |
|---|---:|
| Selling price | Rs. 999 |
| Estimated variable cost | Rs. 720 |
| Contribution per pouch | Rs. 279 |
| Contribution margin | 27.9% |

## First Batch Metrics

| Metric | Value |
|---|---:|
| Batch size | 50 kg |
| Revenue generated | Rs. 47,500 |
| Variable cost | Rs. 36,000 |
| Gross profit | Rs. 11,500 |

## Reinvestment

Rs. 37,500 from first-batch revenue was reinvested into the second production cycle for additional ingredients and rebranded labels.

## Interpretation

The first batch validated demand and confirmed that the venture could generate contribution margin while operating through an asset-light production model. The second cycle was prepared for Sheikhupura retail placement and Lahore market expansion.
